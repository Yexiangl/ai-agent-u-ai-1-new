## 1. 总体结论

AI 助手页已验证的视觉规范（StatusHero + SettingGroup + SettingRow + ActionCluster）**适合分阶段推广，但不应盲目统一**。

- **最适合推广**：关于、本地用量概览、助手记忆（信息/状态/配置类）
- **适合 Hero + 保留原结构**：首页、能力中心、文件库
- **仅轻改视觉**：摸鱼中心、教程、AI 对话
- **不套设置页规范**：AI 对话（聊天优先）、摸鱼中心（保持轻松）

**先做回归再推广**：建议**先执行 TASK-042H**（AI 助手页回归测试），确认 B-F 改造稳定后，再推广到其他页面，避免规范未固化就铺开。

## 2. AI 助手页已验证视觉规范

### 组件

| 组件 | 用途 |
|---|---|
| StatusHero | 顶部状态总览，rounded-3xl + gradient + 状态 badge + 主操作 |
| SettingGroup | macOS Settings 分组容器，rounded-2xl + divide-y |
| SettingRow | 设置行，左 label/desc + 右 value/action + tone dot |
| ActionCluster | 按钮统一排列 |

### 风格令牌

- 圆角：Hero `rounded-3xl`，分组 `rounded-2xl`
- 背景：`bg-gradient-to-br from-card via-card to-muted/20`
- 阴影：`shadow-sm`
- 状态：badge + `h-2 dot`，tone success/warning/danger/muted
- 间距：Hero `p-6 space-y-4`，行 `py-2.5`
- 字号：Hero 标题 `text-2xl font-bold`，行 `text-sm`
