# v0.3.0 内部测试版更新说明

日期：2026-05-29 | 版本：v0.3.0-internal | 状态：内部测试

---

## 面向用户版

### 1. 一键启用 AI 助手

粘贴模型访问密钥后，点击"一键启用 AI 助手"，系统会自动完成：
- 保存配置
- 启动本地服务
- 检查是否可以开始对话

平均只需 10-20 秒，无需了解任何技术细节。

### 2. AI 助手状态更清楚

- **状态卡**：一目了然看到 AI 助手是否已连接、当前模型、密钥是否配置
- **本地服务诊断**：查看本地服务、模型接口、对话接口状态，异常时显示修复建议
- **一键打开控制台**：使用 OpenClaw 官方方式打开本机控制台，可处理本机认证
- **一键启动本地服务**：通过按钮即可启动，无需终端操作

### 3. 能力中心更完整

- **能力排行**：浏览内置能力目录，按热门/趋势/新上架/高风险筛选
- **信息透明**：安装前能看到能力来源、风险等级、权限要求和原生安装命令
- **安装状态**：安装/卸载有分阶段进度和反馈，可确认、可取消
- **已安装管理**：统一查看和管理所有已安装能力

### 4. 本地用量概览

- 显示基于模型接口返回的真实 Token 统计
- 会话数、消息数、Token 用量一目了然
- 数据来自本机历史会话，非服务端账单

### 5. 助手记忆

- 读取 OpenClaw 工作区记忆文件（工作人格、用户偏好、代理规则等）
- 只读查看，内容经过安全脱敏
- 不修改任何记忆文件

### 6. 摸鱼中心

- **今日状态**：给今天起个轻松但准确的名字
- **AI 桌宠**：生成一个陪你工作的虚拟小搭子
- **休息任务**：3 分钟健康休息建议
- **随机冷知识**：工作间隙了解一点反常识
- **今日成就**：把小进展变成一枚成就徽章
- 所有按钮只生成提示词填入 AI 对话输入框，不会自动发送

### 7. 体验优化

- 页面切换有轻量过渡动画
- 卡片 hover 视觉反馈统一
- AI 对话页空状态引导更自然，消息有轻量进入动画
- Toast 轻反馈（成功/失败/警告/信息）
