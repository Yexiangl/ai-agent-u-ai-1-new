## 一、模块定位

摸鱼中心是 Windows / macOS 桌面 AI Agent 工作台里的轻量休息模块。

### 一句话定位

**工作间隙，用 AI 给你一点轻松反馈、短任务、桌宠陪伴和小成就。**

### 不是

- 游戏中心
- 短视频替代品
- 抽卡/氪金系统
- 社交排行榜
- 医疗/心理诊断工具
- 常驻桌宠进程
- 系统通知中心
- Skill Center 的分类页

### 产品心智

工作间隙轻松一下，不耽误正事。用户在主工作台做核心任务（对话、文件处理、能力中心），切到摸鱼中心做 3 分钟以内的小休息。

## 六、首批功能设计

### 6.1 今日状态卡

| 字段 | 内容 |
|---|---|
| **标题** | 今日状态 |
| **说明** | AI 根据你的感受生成娱乐化工作状态 |
| **输入** | 今天感觉怎么样：充满能量 / 正常运转 / 有点疲惫 / 急需摸鱼 |
| **按钮** | 生成今日状态 |
| **行为** | 填入 prompt → 跳转 Agent 对话页 → 不自动发送 |
| **免责** | 娱乐化状态总结，不是医学或心理诊断 |

**Prompt 草案：**

"请用轻松幽默的方式帮我生成一个今日工作状态卡。我今天的感受是：{mood}。请包含：状态名称、状态描述、适合做的事、不适合做的事、一个 10 分钟收尾建议、一句轻松吐槽。注意：这是娱乐化状态总结，不是医学或心理诊断。"

### 6.2 今日休息任务

| 字段 | 内容 |
|---|---|
| **标题** | 今日休息任务 |
| **说明** | 3 分钟内的健康休息小任务 |
| **输入** | 你在哪里：办公室 / 家里 / 咖啡馆 / 通勤路上 |
| **按钮** | 生成休息任务 |
| **行为** | 填入 prompt → 跳转 Agent 对话页 → 不自动发送 |

**约束：** 不刷短视频、不沉迷、不影响正事。

**Prompt 草案：**

"请给我一个 3 分钟以内可以完成的健康休息任务。我在：{env}。要求：不刷短视频、不沉迷、不影响正事，最好能放松眼睛、肩颈或情绪。请输出任务步骤、预计用时、为什么有用、一句吐槽。"

### 6.3 AI 桌宠

| 字段 | 内容 |
|---|---|
| **标题** | AI 桌宠 |
| **说明** | 创建属于你的 AI 桌宠角色设定 |
| **输入** | 选择性格：毒舌吐槽 / 可爱治愈 / 认真学霸 / 佛系躺平 |
| **按钮** | 生成我的桌宠 / 让它吐槽我一句 / 让它鼓励我一句 |
| **行为** | 填入 prompt → 跳转 Agent 对话页 → 不自动发送 |

**第一版不做：** 常驻桌宠进程、桌面悬浮窗、定时弹窗、动态动画。

**Prompt 草案：**

"请帮我设计一个 AI 桌宠角色。性格：{style}。请包含：名字、性格描述、口头禅（2-3句）、喜欢的东西、讨厌的东西、工作时会怎么陪我、摸鱼时会怎么吐槽我。风格轻松可爱但不要太幼稚。"

### 6.4 随机冷知识

| 字段 | 内容 |
|---|---|
| **标题** | 随机冷知识 |
| **说明** | 30 秒内看完的有趣冷知识 |
| **按钮** | 换一个冷知识 |
| **行为** | 填入 prompt → 跳转 Agent 对话页 → 不自动发送 |

**Prompt 草案：**

"请给我一个 30 秒内能看完的有趣冷知识。要求：有一点反常识，不太长。请输出：标题、冷知识内容、为什么有趣、一句轻松吐槽。"

### 6.5 今日成就

| 字段 | 内容 |
|---|---|
| **标题** | 今日成就 |
| **说明** | 根据你完成的事情生成趣味成就徽章 |
| **输入** | 今天完成了什么 |
| **按钮** | 生成今日成就 |
| **行为** | 填入 prompt → 跳转 Agent 对话页 → 不自动发送 |

**第一版不做：** 成就收藏/历史/排行榜/分享。

**Prompt 草案：**

"请根据我今天完成的事情，帮我生成 3 个有趣的成就徽章。请包含：徽章名、稀有度（普通/稀有/传说）、获得条件、吐槽说明。风格轻松幽默，不要太夸张。今天完成了：{done}"

## 七、与 Agent 对话的关系

### 统一行为

所有按钮：点击 → 填入 prompt → 跳转 Agent 对话页 → **不自动发送**

### 约束

- 不新增 AI 调用链路
- 不改 run store
- 不自动发送
- 不自动读取文件
- 不自动读取聊天记录
- 不执行外部 Skill/Plugin
- 不触发 Skill 安装/卸载

与内置工作流行为一致，但入口独立，产品心智不同。

## 九、安全与内容边界

### 不得

| 禁止项 | 原因 |
|---|---|
| 医学诊断 | 不是医疗产品 |
| 心理诊断 | 不是咨询产品 |
| 焦虑/抑郁测试 | 可能误导 |
| 治疗建议 | 法律责任 |
| 诱导沉迷 | 产品伦理 |
| 赌博/博彩 | 法律禁止 |
| 成人内容 | 内容安全 |
| 抽卡氪金 | 产品定位 |
| 排行榜攀比 | 与轻量定位冲突 |
| 自动通知 | 打扰用户 |
| 定时提醒 | 同上 |
| 常驻桌宠进程 | 第一版不做 |
| 读取文件 | 隐私安全 |
| 读取隐私数据 | 同上 |
| 读取 .env | 安全边界 |
| 输出 Token | 安全边界 |
| 执行外部命令 | 安全边界 |
| 安装/卸载 Skill | 不是 Skill Center |

### 必须

- "今日状态"必须标明：娱乐化状态总结，不是医学或心理诊断
- "AI 桌宠"不做常驻进程
- "今日成就"不做长期排行
- 所有 prompt 跳转不自动发送
