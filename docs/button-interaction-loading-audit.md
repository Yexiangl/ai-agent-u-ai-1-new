## 6. 统一交互规范

### 6.1 异步按钮模板

```
idle:     文案 + 无 spinner
loading:  Loader2 h-4 w-4 animate-spin + "正在..." + disabled
success:  inline 绿色提示 或 toast
failure:  inline 红色提示
```

### 6.2 Spinner 尺寸

| 场景 | 尺寸 |
|---|---|
| 普通按钮内 | h-4 w-4 |
| 小按钮 (size="sm") | h-3.5 w-3.5 |
| 图标按钮 (size="icon") | h-4 w-4 |
| 全页 loading | h-5 w-5 |

### 6.3 复制反馈

统一为：
- 图标变 Check（绿色）持续 2s，然后恢复
- 或 showToast("已复制", "success")

### 6.4 Toast 使用场景

| 场景 | 用 Toast |
|---|---|
| 复制成功 | ✅ |
| 保存文件成功 | ✅ |
| 安装完成 | ✅（补充 inline） |
| 卸载完成 | ✅ |
| 打开控制台失败 | ✅ |

| 场景 | 用 Inline |
|---|---|
| 配置启用失败 | ✅ |
| 本地服务启动失败 | ✅ |
| AI 对话失败 | ✅ |
| 安装失败 | ✅ |

### 6.5 Disabled 规则

| 状态 | 视觉 |
|---|---|
| 异步 loading | disabled + opacity-50 + cursor-not-allowed |
| 条件不满足 | disabled + opacity-50 |
| 功能未开放 | disabled + outline + "暂未开放" |
