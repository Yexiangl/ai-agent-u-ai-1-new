# AI Agent Workspace 发布前验收清单

> 用于每次打包或给客户试用前逐项测试。完成后请勾选对应条目。

## 1. 基础启动

- [ ] `npm run build` 通过
- [ ] `cargo check` 通过
- [ ] `npm run dev` 可启动
- [ ] 首次启动不白屏
- [ ] App 关闭后重新打开正常
- [ ] 深色/浅色模式切换正常

## 2. Onboarding

- [ ] Hermes 未安装时显示友好提示（非白屏/崩溃）
- [ ] Hermes 已安装但 API Server 未运行时显示引导
- [ ] Token 为空时不能错误写入
- [ ] Token 错误时测试失败提示清楚
- [ ] Token 正确时测试成功
- [ ] 选择模型后可以应用到 Hermes
- [ ] 跳过 Onboarding 进入不会破坏已有配置
- [ ] Onboarding 完成后配置持久化，重启不失

## 3. Hermes 管理

- [ ] Hermes CLI 检测正常（版本号、路径正确）
- [ ] 本地 Hermes API Server 检测正常（running / models 列表）
- [ ] 模型卡片选择正常（deepseek-v4-flash / pro / kimi-k2.6）
- [ ] 应用到 Hermes 步骤式 UI 正常
- [ ] 应用成功后 model / provider / baseUrl 写入 config.yaml 正确
- [ ] Token 不出现在 UI、日志、错误信息中
- [ ] 售后诊断入口可访问，不包含 Token
- [ ] baseUrl / provider 不在普通 UI 暴露

## 4. Agent 对话

- [ ] 普通消息可发送

- [ ] 回到底部按钮位置正确且可用
- [ ] 中文输入法 Enter 选字不误发送
- [ ] Shift + Enter 换行正常
- [ ] 生成完成后快速回到就绪状态
- [ ] Hermes 回复不会丢失或空白
